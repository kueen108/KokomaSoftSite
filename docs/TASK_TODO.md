# Ungyeol Web Mini Services TODO

KokomaSoftSite에 운결 앱의 핵심 기능을 웹 미니 서비스로 제공한다. 웹은 무료 맛보기와 SEO 유입을 담당하고, 정밀 풀이/히스토리/사진 관상/복채 사용은 앱으로 전환시키는 구조를 유지한다.

## 원칙

- [ ] 웹 결과는 클라이언트 사이드 룰 기반 미니 풀이로 구현해 AI 비용과 서버 비용을 만들지 않는다.
- [ ] 사진 업로드가 필요한 관상은 웹에서 원본 사진을 받지 않고, 앱 설치 유도형 안내/체험 페이지로 제공한다.
- [ ] 각 기능 페이지는 고유 title, description, canonical, sitemap 항목을 가진다.
- [ ] 모든 결과 화면에는 운결 앱에서 더 자세히 보기 CTA를 제공한다.
- [ ] 검색 유입용 문구는 자동 대량 생성 블로그가 아니라 사용자가 실제로 조작할 수 있는 도구형 페이지 중심으로 구성한다.

## 구현 체크리스트

### 1. 공통 기반

- [ ] 운결 웹 미니 서비스 목록과 SEO 메타 데이터를 정의한다.
- [ ] 결과 생성에 사용할 공통 유틸리티와 deterministic seed 함수를 만든다.
- [ ] 운세, 상대 속마음, 답장 추천, 꿈해몽, 타로, 인연/궁합, 자유질문, 관상 안내용 룰 기반 결과 생성기를 만든다.
- [ ] 공통 서비스 페이지 컴포넌트를 만들어 입력 폼, 결과 카드, 앱 전환 CTA를 재사용한다.

### 2. 기능 페이지

- [ ] `/ko/fortune/` 오늘 운세 미니 풀이 페이지를 추가한다.
- [ ] `/ko/hidden-feelings/` 상대 속마음 미니 풀이 페이지를 추가한다.
- [ ] `/ko/reply/` 답장 추천 페이지를 추가한다.
- [ ] `/ko/dream/` 꿈해몽 미니 풀이 페이지를 추가한다.
- [ ] `/ko/tarot/` 타로 3장 미니 리딩 페이지를 추가한다.
- [ ] `/ko/compatibility/` 인연/궁합 미니 풀이 페이지를 추가한다.
- [ ] `/ko/free-question/` 자유질문 미니 풀이 페이지를 추가한다.
- [ ] `/ko/face-reading/` 관상 기능 안내 및 앱 전환 페이지를 추가한다.

### 3. 사이트 연결

- [ ] 한국어 홈에 운결 미니 서비스 섹션을 추가한다.
- [ ] 운결 앱 상세 페이지에서 웹 미니 서비스로 진입할 수 있게 한다.
- [ ] 각 미니 서비스 페이지에서 Google Play 운결 링크로 전환되게 한다.

### 4. SEO 및 배포

- [ ] 사이트맵에 한국어 미니 서비스 URL을 포함한다.
- [ ] 각 페이지의 canonical, Open Graph, Twitter card, JSON-LD를 확인한다.
- [ ] `npm run build`를 통과시킨다.
- [ ] Cloudflare에 배포하고 라이브 URL을 검증한다.
- [ ] 모든 체크박스를 완료 상태로 업데이트한다.
